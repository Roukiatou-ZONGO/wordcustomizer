# WordCustomizer

Traitement de texte libre et open source basé sur LibreOffice.
Interface noir, violet et blanc — compatible Microsoft Word (.docx).

## Téléchargement

| Fichier | Lien |
|---------|------|
| Fichiers du logiciel (tar.gz) | [Télécharger sur Google Drive]https://drive.google.com/file/d/1xrb3ti4T2NrBtuTPwE9eXRV4XPrnNCVh/view?usp=drive_link |
| Script d'installation | [Voir les Releases GitHub](https://github.com/Roukiatou-ZONGO/core/releases) |

## Installation (Linux / WSL)

1. Téléchargez les deux fichiers ci-dessus
2. Mettez-les dans le même dossier
3. Ouvrez un terminal et tapez :

```bash
    bash install-wordcustomizer.sh
```

4. Lancez WordCustomizer :

```bash
    wordcustomizer
```

## Fonctionnalités

- Édition de documents Word (.docx)
- Interface moderne noir, violet et blanc
- Compatible Microsoft Word
- Styles et thèmes personnalisables
- Macros et automatisation
- 100% gratuit et open source
